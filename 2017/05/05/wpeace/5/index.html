鉴别塑料大米的六种方法

发表于 2017-05-05 | 分类于 世界和平 | 阅读
字数: 474 | 阅读时长 ≈ 2 分钟

　　据外媒报道称，国内有少数不法商贩制作的塑料大米流通到了很多国家，其中还包括非洲的加纳等国家。韩国知名媒体报道称，“中国以无所不有、无所不造而著称，现在有报道称使用塑料制作的大米在全球市场上流通。”
　　据报道，这种“塑料大米”是是将土豆压制成普通大米的外形后，使用人工树脂固化而成。这种大米煮熟之后，仍然保持着硬度，并且会引起严重的健康问题。专家警告称，可能会有大量此类“塑料大米”流入非洲市场。

要鉴别这类塑料大米，可使用以下六种方法：

  1. 捣碎法
    　　用研钵将少许米粒研碎，如果米粉呈白色则为正常大米；如果颜色发黄则可能有问题。

  2. 火烧法
    　　大家都知道塑料烧焦是什么味道。因此，可以用打火机烧一下米粒试试。如果能够点燃并且闻起来有塑料烧焦的问道，则是假米。

  3. 水洗法
    　　取一杯清水，倒一勺米进去，然后搅拌。如果米粒沉到水底，则没有问题。如果米粒漂浮在水面上，则说明是假米。

  4. 真菌法
    　　将米粒煮开，并放置在温度较高的地方两到三天。如果米粒没有生霉，则说明是假米。因为塑料是不会因温度和湿度的变化而发霉的。

  5. 煮开法
    　　在煮米饭的时候，观察一下水面是否会有一厚层米饭出现，如果是的，则可判断为塑料大米。

  6. 热油法
    　　直接将少许米粒放入滚烫的油中，如果是塑料的，它会融化，并粘在一起或者粘在锅底上。
