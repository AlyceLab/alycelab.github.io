氚3H（T）甘斯应用 :

3H跟镍作用下释放巨大热量，这个热量可以做热空调， 不耗电的热空调；
3H放到电池里面去，能够提高电池产生巨大电量，3H放到电池属于核电池；
3H的巨大磁场，能够让人恢复年轻化，因为你的人体由挺都的H元素组合成，3Ｈ能够作用到人体；
3H加入双核反应器里面和其他甘斯组合会创造飞升；
3H是制造物质及食物的源材料。

氚3H甘斯的制造方法1：（可乐瓶法）

​ 氚甘斯用途广泛，可以利用其能量的场体进行飞升，发电等的用途。但制作氚甘斯不同于以往的制作方法，首先它不是露天开放的，它是在密封的环境下进行，因为氢气一但跑出去了，就产生不了氚甘斯，氚是氢的n种同位素之一。

  1. 正宗可口可乐600mL，1000mL几瓶，18mm直径45mm长空心铜管4根和螺旋铜线4个纳米化，双酚A型环氧树脂；
  2. 用酒精和蒸馏水清洗干净瓶子内外及瓶盖（可乐用于浸泡PET塑料， 瓶口和内部不可用嘴手触碰）；
  3. 在瓶身上按需开孔，洗干净；
  4. 配好NaOH和KOH（1：3）的冷碱液浸涂满瓶子全身，凉干静置3小时后再做一次，然后进行脉冲；
  5. 将瓶子浸没于冷碱液中48小时，用水冲洗晾干，再重复浸泡冲洗前面的过程1遍；
  1. 将铜管和铜圈按装上，并用树脂胶密封好不漏气，大瓶底部进行固定，以方便浸入水中；
  2. 将冷碱液倒入大瓶子，液面底于横放时的瓶口，再把压紧卷好的铝箔小球2个装入瓶内，晃动碱液，散发出热量，然后把瓶子的一半浸入箱子的冷水中，勿使瓶子融化，然后尽量抽去瓶子内的空气；
  3. 静置2～10天，可以在最上面的瓶壁上看到紫色、黄色或黑灰色液滴，说明成功制得氕氘氚甘斯。

注意事项：

  1. 碱液要在瓶外配好并减至常温；
  2. 碱液配比：100毫升蒸馏水中加3至5克氢氧化钠，氢氧化钾9至15克为宜；
  3. 碱液不可满过瓶盖，铜圈不可碰到碱液；
  4. 铜管下口尽量与瓶子平，不可太长，上口可以长一些，放便气体的走动。



氚甘斯的制造方法2：（CH3法）

  1. 带盖CO2甘斯制造包一套，200ml石英玻璃瓶一个；

  2. 玻璃瓶中装满带有CH3上清水的蒸馏水，再放入余留空间尽量少的CO2制造包中；

  3. 一星期后，再换新的CO2制造包，玻璃瓶不变；

  4. 直到玻璃瓶里的水逐渐变为黑灰色。

氚甘斯的制造方法3：（重水法）

  1. 在CO2甘斯制造过程中，逐滴滴入重水，间隔在半分钟以上；
  2. 水中逐渐变为浅蓝色或黑灰色，即是氚或氘甘斯。

氚甘斯的制造方法6：（PET法）

  1. 大超市购买正宗的可口可乐3000mL以上，倒入较多的剪碎PET塑料，密封6天以上；
  2. 把PET塑料用蒸馏水漂洗干净；
  3. 对PET塑料象做铜纳米涂层一样，加碱片和钾片进行纳米涂层，浸泡2～3天，蒸3次以上；
  4. 取适量的50～100度蒸馏水，逐片放入碱片，直到水面出现很多泡泡不消失；
  5. 把PET塑料放入反应釜中并倒入新的碱液，透明碱液会逐渐变为浅蓝色；
  6. 变浅蓝色后，给反应釜加压可以生成氚和氘甘斯；
  7. 提取出紫色或黑灰色溶液进行沉淀清洗。
