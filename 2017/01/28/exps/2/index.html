保罗的节电95%双能量单元系统-V1

发表于 2017-01-28 | 分类于 实验相关 | 阅读
字数: 512 | 阅读时长 ≈ 2 分钟

作者：凯史基金会多哥分会——保罗
原文：保罗的博客
译文：艾丽斯

这个双能量单元系统有三个电容。通电之后立即达到节电80%，一周后节电达到95%。

在我们的系统当中，两个能量单元结构完全相同，呈镜面对称。

#系统组成：

  1. 三个简易电池电容（中心电极＋浸泡过硼砂溶液的耐火纸；混合甘斯，包括：氧化铜甘斯、铅甘斯、二氧化碳甘斯、甲基甘斯、黄铜甘斯、青铜甘斯、纳米涂层铝箔＋纳米线圈，美规14号铜线，18圈逆时针方向绕制）。中心电极并联到零线，线圈并联到火线）

  2. 火线接到每个能量单元的输入端。

  3. 每个单元的输出端连接到分离式电池电容的中心棒；这些电容的制作方法同第一步。

  4. 反馈线圈接到能量单元的输入端来进行反馈。

  5. 线圈后端是输出。

  6. 第一个的输出连接到第二个的输入，且第二个的输出连接到第一个的输入。

  7. 两个单元的其中一个输出连接到火线和负载。

  8. 从简易电池电容出来的零线直接连接到负载上。

结论：

  • 在第一轮测试中，我们将一个干燥机连接到能量单元上。开始的消耗功率是408瓦。过了几分钟后，消耗功率下降到33.4瓦。

  • 在第二轮测试中，我们将一个冰箱连接到能量单元上，制造商信息显示这个冰箱的功率是940瓦。通常连接它消耗功率是800瓦，而接到能量单元后消耗功率变为了120瓦。

  • 我不清楚我们家的电线被纳米化了没有。

译注：关于此双能量单元的制作方法，请参见本实验室的另一篇译文：《节电95%的双能量单元系统》。

(完）
